# Aashi Goyani Portfolio

Minimal static portfolio website for Aashi Goyani.

## Run locally

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173` in a browser.

## Deploy with GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository, open **Settings > Pages**.
3. Set the source to **Deploy from a branch**, then select `main` and the `/ (root)` folder.
4. Save. GitHub will publish the site at `https://aashigoyani.github.io/<repository-name>/`.
